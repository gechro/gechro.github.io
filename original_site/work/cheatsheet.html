Ruby/JS Iterator Cheatsheet

Ruby

#each

a = [0,1,2,3,4]
a.each do |number|
  puts number
end

=>0
=>1
=>2
=>3
=>4

#each is an Enumerable method that can be used on arrays and hashes. In the example above, #each is being called on the array 'a'. The #each method goes through each element of the array, and runs the code that follows, for each element. Alternatively, each can be called with a code block, using curly brackets '{}', instead if do...end:

a = [0,1,2,3,4]
a.each {|number|  puts number}

=>0
=>1
=>2
=>3
=>4

while

num = 0
while num < 10
  puts num
  num+=1
end

=>0...9

#while is a method that loops a block of code as long as the condition is true.

JS

for ... in ...

var a = [0,1,2,3,4]
for (number in a){
  console.log(number);
};

=>0
=>1
=>2
=>3
=>4

for...in... iterates over the propertiess of an object, and runs the code in the block for each property. 'for' is followed by the variable/property 'in' the object that is being iterated over. You can use "for each x in y" to iterate over the values of "y".
